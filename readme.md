# Instant Meeting（即时会议）
### 前端 React + MUI + simple-peer
```bash
pnpm i
pnpm electron:rebuild
pnpm dev
# robotjs的安装比较麻烦，安装以后执行以下npx electron-rebuild
```

### 后端 node + express + socket.io
```bash
pnpm serve
```
